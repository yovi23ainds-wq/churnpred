# Churn Prediction Dashboard - Frontend

React + Vite + Tailwind CSS frontend for customer churn prediction.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Single-page form (18 fields)
- Real-time predictions
- Interactive visualizations
- Action modals (Email, Offer, Profile)
- Responsive design

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Axios
